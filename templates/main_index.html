NQSync

A Next-Gen Network Simulator with Quantum Edge

NQSync is a web-based network simulator inspired by ns-3, built with the future of quantum-classical hybrid networking in mind. Create and test network topologies in real-time using an intuitive drag-and-drop interface.

Features

Visual Topology Builder

Drag nodes, connect links

Quantum-Compatible

Includes protocols like CoW and DPS

Real-Time Simulation

Detailed Simulation Metrics
